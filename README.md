# FakeStore App

## Overview

This project is a React application that uses the FakeStore API to perform CRUD operations on products.

## Features

- View all products
- View product details
- Add a new product
- Edit an existing product
- Delete a product
- React Router navigation
- Axios API requests
- Bootstrap styling

## Technologies Used

- React
- React Router DOM
- Axios
- Bootstrap
- Vite
